# Screenshot Capture Checklist

Drop captured screenshots in this folder using the exact filenames below. Recommended device: **iPhone 15 Pro (1179×2556)** or any 3:6.5 aspect ratio Android device.

| File | Screen | Notes |
|---|---|---|
| `home.png` | Home / dashboard | Show total balance + 2–3 wallet cards |
| `transactions.png` | Transaction list | At least 6–8 transactions of mixed types |
| `slip-parser.png` | Add-transaction flow with slip preview + extracted fields | Hero feature — make this look polished |
| `budgets.png` | Budgets list | At least one budget near/over limit to show status colors |
| `analytics.png` | Analytics | Category pie + monthly trend visible |
| `wallets.png` | Wallets list + balances | 3+ wallets of different types |
| `lock-screen.png` | Biometric unlock screen | Showcases security feature |

## Optional extras

- `categories.png` — Category editor
- `export.png` — Export menu
- `settings-lao.png` — Settings screen in Lao (showcases i18n)
- `slip-parser-result.png` — Side-by-side photo + parsed fields

## Tips

- Use device simulators for clean status bars (`Cmd+Shift+H` to hide notch shadow on iOS).
- Populate with realistic but anonymous data — avoid real bank names / amounts.
- Compress PNGs with TinyPNG before committing.
